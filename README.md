# HR Attrition Dashboard

Analyzed HR dataset of 1,400+ employees to identify attrition trends and key factors like job role, satisfaction, and salary. Designed interactive dashboards with KPIs such as Attrition Rate, Average Age, and Department-wise Trends. Provided actionable insights to improve retention and engagement.
